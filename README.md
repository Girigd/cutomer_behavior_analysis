# 🛍️ Customer Shopping Behavior Analysis

## 📌 Overview

This project analyzes customer shopping behavior to identify purchasing patterns, customer preferences, and business insights. The analysis follows a complete data analytics workflow, starting from data preprocessing in Python to interactive dashboard creation in Power BI.

The project demonstrates practical skills in **Python, MySQL, SQL, Power BI, and data visualization** while presenting insights in a professional report and presentation.

---

## 📂 Dataset

The dataset contains customer shopping information, including:

* Customer demographics
* Product categories
* Purchase amounts
* Payment methods
* Shopping frequency
* Discounts and promotions
* Customer ratings

---

## 🛠️ Tools & Technologies

* **Python** – Data loading, cleaning, and Exploratory Data Analysis (EDA)
* **Pandas** – Data manipulation and preprocessing
* **MySQL Workbench** – SQL querying and business analysis
* **Power BI** – Interactive dashboard development
* **Gamma** – Presentation creation
* **Microsoft Word** – Project documentation and reporting

---

## 📈 Project Workflow

### 1. Data Loading

* Imported the dataset into Python.
* Verified data types and dataset structure.

### 2. Exploratory Data Analysis (EDA)

* Explored customer demographics and purchasing behavior.
* Identified trends and relationships using visualizations.
* Examined missing values and data quality.

### 3. Data Cleaning

* Handled missing values.
* Removed duplicate records.
* Standardized data formats.
* Prepared the dataset for analysis.

### 4. SQL Analysis

* Imported the cleaned dataset into MySQL Workbench.
* Wrote SQL queries to answer business questions.
* Used filtering, grouping, sorting, aggregate functions, and joins (where applicable) to generate insights.

### 5. Power BI Dashboard

* Connected the processed data to Power BI.
* Built interactive dashboards using KPIs, charts, slicers, and filters.
* Designed visualizations for easy business interpretation.

### 6. Reporting & Presentation

* Documented the complete analysis in a project report.
* Created a presentation using Gamma to summarize findings and recommendations.
